									Проект "Battle City".
Проект является клоном оригинальной игры Battle City на Dendy, в нём реализованы не все функции, но в будущем это будет дорабатываться.
Суть проекта - реализовать клон игры, тем самым получить опыт по созданию игр.
Файлы проекта и их назначения:
1.main.py - основной файл проекта, через него запускается игра.В нем реализован начальный экран игры, конечный экран, вывод статистики и запуск игры.
2.player.py - файл содержит в себе реализацию функций игрока.
3.enemy.py - файл содержит в себе реализацию функций противника.
4.bullet.py - здесь раелизован класс пули, пуля может присваиваться как игроку так и противнику.
5.bonus.py - файл отвечающий за спавн рандомного бонуса в случайной свободной клетке.
6.field.py - файл отвечает за начальное отображения клеток поля.
7.base.py - файл отвечает за "базу" которую должны уничтожить противники.
8.vars.py - файл содержит в себе переменные, которые импортируются во все файлы проекта и используются там.
9.tile.py - файл имеет класс Tile который создает спрайт нужной клетки.
10.spawn.py - здесь находится классс Spawn который создаёт противников.
